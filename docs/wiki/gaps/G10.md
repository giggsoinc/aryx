## What Was Built

- **Band redesign (DEC-002, data-driven from G9's sweep):** four env-tunable
  thresholds — `ARYX_ER_AUTO_MERGE` 0.92 / `ARYX_ER_ADJUDICATE` 0.90 /
  `ARYX_ER_REVIEW` 0.75 / reject below. Pairs in [0.75, 0.90) were silently
  lost before; they now queue for human review as non-merge (conservative).
- **Migration 0020** — `aryx_adjudication`: every row including LLM-decided
  `auto_llm` ones is a labeled (pair, score, verdict) training example — the
  data moat. Quarterly export recommended.
- **`resolution/review_queue.py`** — `ReviewSink` protocol, store sink,
  `apply_decision()`: approval unions the two entities post-hoc (members
  move, attributes merge, `updated_at` touched → feeds G8's dirty set);
  rejection records the label.
- **`store/adjudication_store.py`** — queue CRUD, stats (incl. human/LLM
  agreement rate — the sales stat), `merge_entities_of`.
- **API** — `GET /adjudication` (paged), `POST /adjudication/{id}/decide`,
  `GET /adjudication/stats`. Behind G4's ApiKeyMiddleware.
- **LLM failure = fail-to-human (DEC-003):** adjudication exceptions queue
  the pair as pending instead of propagating or silently passing.

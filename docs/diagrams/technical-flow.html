Same-origin /api proxy

Browser only talks to :3000. Next.js rewrites /api/* to http://api:8000/* on the docker network via ARYX_API_URL_INTERNAL. No CORS, no build-time host coupling — same bundle runs in dev, EC2, or any reverse proxy.
